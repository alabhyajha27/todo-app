# Todo List Web App

A simple web application that allows users to create and manage daily tasks.
The application helps users stay organized by adding, updating, and deleting tasks.

## Features

* Add new tasks
* View all tasks in one place
* Mark tasks as completed
* Update existing tasks
* Delete tasks
* Clean and simple user interface

## Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS
